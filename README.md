# منصة ذكاء العملات الرقمية - Crypto Intelligence Platform

**منصة حديثة وشاملة لتحليل ومراقبة أسواق العملات الرقمية**

## 🚀 المميزات الرئيسية

### 📊 الأسعار المباشرة
- تتبع أسعار العملات الرقمية في الوقت الفعلي
- الترتيب والبحث والتصفية
- إضافة العملات للمفضلة

### 📰 الأخبار الرسمية
- أخبار رسمية للمشاريع والعملات
- تحديثات وشراكات جديدة
- إطلاق خدمات وميزات جديدة
- تصنيف الأخبار (عاجل - رسمي - أخبار السوق)

### 📈 حجم التداول (Volume)
- مراقبة أحجام التداول الحية
- الربط بين حجم التداول وحركة السعر
- مؤشرات نشاط السوق

### 🔍 التحليل الفني
- **مؤشر قوة النسبية (RSI)**: تحديد مناطق الشراء والبيع الزائد
- **مؤشر MACD**: تحديد الاتجاهات والقوة
- **المتوسطات المتحركة (MA)**: تحديد الاتجاه العام
- **مستويات الدعم والمقاومة**: نقاط انعكاس السعر
- **إشارات البيع والشراء**: إشارات تداول موثوقة

### 🐋 نشاط الحيتان
- تتبع محافظ الحيتان (المحافظ الضخمة)
- عمليات الشراء والبيع الكبيرة
- معرفة اتجاهات المتداولين الكبار

### 🔔 التنبيهات الذكية
- **تنبيهات الأسعار**: عند ارتفاع أو انخفاض السعر
- **تنبيهات حجم التداول**: عند ارتفاع مفاجئ في الحجم
- **التنبيهات الفنية**: عند ظهور إشارات تقنية مهمة
- **تنبيهات الأخبار**: عند صدور أخبار رسمية
- **تنبيهات نشاط الحيتان**: عند حركات الحيتان

### ⭐ المفضلة
- قائمة شاملة للعملات المفضلة
- متابعة سريعة للعملات الهامة

### ⚙️ الإعدادات
- **اختيار اللغة**: العربية والإنجليزية (RTL/LTR)
- **المظهر**: وضع داكن (مفضل) ووضع فاتح
- **العملة**: دولار - يورو - درهم - جنيه
- **إدارة التنبيهات**: تخصيص التنبيهات المفعلة

## 📋 المتطلبات التقنية

- **لا يوجد تبعيات**: HTML + CSS + JavaScript فقط
- **لا يوجد Build Tools**: يعمل مباشرة في المتصفح
- **بدون Backend**: جميع البيانات من Free APIs
- **Responsive Design**: يعمل على جميع الأجهزة

## 🔧 APIs المستخدمة

1. **CoinGecko API** - أسعار وبيانات العملات
2. **Fear & Greed Index API** - مؤشر الخوف والجشع

## 📊 البيانات المستخدمة

### الأخبار الرسمية
- **تحديثات المشروع**: الإعلانات الرسمية والتطورات
- **الشراكات الجديدة**: التعاونيات والاتفاقيات
- **إطلاق الخدمات**: ميزات وخدمات جديدة
- **المواقع الرسمية**: روابط الموارد الرسمية

### حجم التداول
- **Volume 24h**: إجمالي حجم التداول
- **Volume/Market Cap**: نسبة الحجم إلى القيمة السوقية
- **Buyer/Seller Ratio**: نسبة المشترين للبائعين
- **Exchange Volume**: حجم كل منصة تداول

### التحليل الفني
- **Support & Resistance**: مستويات الدعم والمقاومة
- **RSI (Relative Strength Index)**: قياس قوة الاتجاه
- **MACD**: تقارب/تباعد المتوسطات المتحركة
- **Moving Averages**: المتوسطات المتحركة 20/50/200
- **Trend Strength**: قوة الاتجاه (قوي/ضعيف)
- **Buy/Sell Signals**: إشارات التداول

### نشاط الحيتان
- **Large Transactions**: عمليات كبيرة فوق 1 مليون دولار
- **Whale Wallets**: تتبع محافظ الحيتان المعروفة
- **Buy/Sell Ratio**: نسبة عمليات الشراء للبيع
- **Market Impact**: تأثير الحيتان على السعر

### مؤشرات السوق العامة
- **Bitcoin Dominance**: سيطرة البيتكوين على السوق
- **Fear & Greed Index**: مؤشر نفسية السوق
- **Market Cap**: القيمة السوقية الإجمالية
- **24h Volume**: حجم التداول الكلي
- **Altcoin Movements**: حركات العملات البديلة

## 💡 ملاحظات هامة

### تحركات الأسعار الطبيعية
- تحركات **1-2 سنت في دقائق** = ضوضاء عادية
- **الدقة**: تركيز على الاتجاهات طويلة الأجل
- **التحليل**: الجمع بين السعر والحجم والمؤشرات

### إشارات موثوقة
✅ **قوية:**
- ارتفاع حجم مع ارتفاع السعر
- إشارة تقنية متعددة متسقة
- دعم معروف من الحيتان

❌ **ضعيفة:**
- حركة سعر بدون حجم
- إشارة واحدة فقط
- تقلبات سريعة
